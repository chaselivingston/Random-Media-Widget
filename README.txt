Display a random image from your Media Library in a widget.

== Description ==

This plugin allows you to create a new widget that will display either a random image from your entire Media Library, or a random image from a subset of images organized by tag.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Add the widget to a Widget Area in Appearance > Widgets.
